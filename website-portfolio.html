About the Project

This portfolio site was created using a combination of knowledge gathered from various tutorials. Display has been optimized to fit large screens over 1200px and small phones.

Technology

The site was created using HTML, Sass (with BEM organization), and Javascript. All site design and main site logo were personally created and developed. Animations were created using the Animate.css and Awesome Slider libraries. Code was processed using several npm packages for optimum efficiency of page loads.
  • Sass
  • Vanilla Javascript
  • Responsive Design
  • HTML5
  • npm
  • Front End Development
  • UI/UX Design and Animations
  • Inline SVGs
