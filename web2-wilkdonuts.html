About the Project

This websites creates an online front for the donut truck stand. The site was designed, styled, and coded by myself, including taking and editing of photos as well as writing.

Technology

Photos were edited using Pixlr Editor and layouts were setup using CSS Grid. The following tools were used during the make of this project:
  • Front End Development
  • UI/UX Design
  • CSS + CSS Grid
  • FontAwesome
  • HTML
  • JQuery
  • Javascript
