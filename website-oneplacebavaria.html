About the Project

One Place Bavaria was built out of frustration of never learning about fun things before they happened in a small community in Germany. This site relies on scrapers to pull from a few sites as well as manual entry from anonymous contributors (on public-facing pages) and admin (from the admin dashboard only accessible through login).

Technology

The project was built with a LAMP stack and is hanging out on a Linode server under a custom domain name and necessary SSL certificates. The focus was learnign php and MySQL, so a template was used for the admin pages and that was used as inspiration for desinging the public face of the site. The following tools were used during the making of this site:
  • php
  • MySQL
  • CSS
  • HTML
  • Javascript
  • npm
  • node (puppeteer, express, typeORM)
  • Linux
  • Linode
